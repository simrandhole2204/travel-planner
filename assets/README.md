# Travel Planner Assets

This directory contains app assets:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Placeholder Assets

The current assets are placeholders. Replace them with your actual app branding:

1. **App Icon** (icon.png)
   - Size: 1024x1024 px
   - Format: PNG with transparency
   - Design: Your app logo

2. **Splash Screen** (splash.png)
   - Size: 2048x2048 px (will be resized)
   - Format: PNG
   - Design: App logo centered on white background

3. **Adaptive Icon** (adaptive-icon.png)
   - Size: 1024x1024 px
   - Format: PNG with transparency
   - Design: Foreground layer of your icon

4. **Favicon** (favicon.png)
   - Size: 48x48 px
   - Format: PNG
   - Design: Small version of your icon

## Generating Assets

You can use Expo's asset generation tools:

```bash
npx expo-asset-generator --icon path/to/your/icon.png
```

Or use online tools like:
- https://www.appicon.co/
- https://makeappicon.com/
